Đây là tài liệu test về Node.js và Express.js.

Node.js là một runtime environment cho JavaScript được xây dựng trên V8 JavaScript engine của Google Chrome. 

Các tính năng chính của Node.js:
- Event-driven, non-blocking I/O model
- Hệ sinh thái npm với hàng triệu packages
- Phù hợp cho việc xây dựng API server
- Hỗ trợ microservices architecture

Express.js là web framework phổ biến nhất cho Node.js:
- Minimal và flexible
- Middleware system mạnh mẽ
- Routing system dễ sử dụng
- Hỗ trợ template engines

Ví dụ tạo server đơn giản:
```javascript
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(3000, () => {
  console.log('Server running on port 3000');
});
```

Kết luận: Node.js và Express.js là lựa chọn tuyệt vời cho backend development.
